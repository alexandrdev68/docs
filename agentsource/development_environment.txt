Install vagrant:

 - download and install vagrant package;
 - download vagrant box;
 - install vagrant plugin: 
 	* sudo vagrant plugin install vagrant-vbguest;
 - setup vagrant box "package":
 	* vagrant box add package.box --name agentsource;
